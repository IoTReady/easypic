Usage Instructions

This program prepares the assembly code for a PIC10F200 to program an ADF4160 PLL. The code should be compatible with other micro-controllers from the PIC10F20X family and possibly with other PLLs with the same latch structure and programming method. The generated assembly code is essentially just a text file and can be edited to suit other configurations.

Code Generation

Once you enter the counter values and set other bits as per your required settings, you can generate the assembly code based on a pre-existing template. By default, the program uses a template stored internally. However, you can copy this internal template and store an edited copy with the name "code_template.asm" in the same directory as this executable and your edited template will be used instead.

Building a Hex File

For convenience, you can use the "Save & Build" button to save the generated assembly code and compile it using the MPASMWIN assembler. The generated Hex file is stored in the same directory as the assembly code. The "Save & Build" button is only highlighted if the assembler is automatically located or you select it manually in the Settings tab.

Settings Tab

The Settings tab provides access to some basic options for configuring the PIC and for building the hex file. The default settings should work for most users.

For more details refer to the respective datasheets.
